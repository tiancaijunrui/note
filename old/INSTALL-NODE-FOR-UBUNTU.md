### 步骤

- ```
  cd /etc/web/
  mkdir node
  cd node
  ```

- ```
  wget https://nodejs.org/dist/v9.9.0/node-v9.9.0-linux-x64.tar.xz
  ```

- ```
  xz -d node-v9.9.0-linux-x64.tar.xz
  tar xvf node-v9.9.0-linux-x64.tar
  ```
